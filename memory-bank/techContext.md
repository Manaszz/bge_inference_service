# Tech Context

## Technology Stack
- **Language**: Python 3.11+
- **Framework**: FastAPI, Pydantic v2
- **ML**: FlagEmbedding (BGE), Transformers, PyTorch (CUDA 12.4)
- **Containerization**: Docker, Docker Compose

## Development Setup
- **Venv**: Local `.venv` used for development and testing.
- **Mirror**: Specific versions pinned for closed-contour compatibility (e.g., FastAPI 0.122.0).

## Tooling
- **context7**: Primary tool for fetching library updates and docs.
- **pytest**: For integration and unit testing.
